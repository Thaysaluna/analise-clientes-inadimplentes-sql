# Análise de Clientes Inadimplentes (SQL)

##  Contexto
Este projeto simula um cenário de uma empresa SaaS com cobrança recorrente, com foco na análise de clientes inadimplentes e identificação de risco de churn.

## Objetivo
Identificar:
- Clientes com pagamentos vencidos
- Quantidade de parcelas pendentes
- Status da assinatura
- Possíveis cancelamentos e seus motivos

## Tecnologias utilizadas
- SQL

## Principais análises
- Cálculo de vencimento ajustado por empresa
- Identificação de inadimplência
- Contagem de parcelas pendentes
- Análise de eventos de cancelamento

## Sobre o projeto
Este projeto foi inspirado em análises realizadas em ambiente profissional, com adaptação para um cenário fictício e anonimizado.
